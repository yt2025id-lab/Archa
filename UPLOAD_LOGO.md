# Upload Logo Archa

## Instruksi Upload Logo

Untuk mengganti icon/favicon website, silakan upload file **archa-hitam.png** ke folder berikut:

```
/Users/macbookpro/Documents/Archa/public/
```

### Path Lengkap:
```
/Users/macbookpro/Documents/Archa/public/archa-hitam.png
```

### Spesifikasi Logo:
- **Nama File**: `archa-hitam.png`
- **Format**: PNG dengan transparent background (recommended)
- **Ukuran**: Minimal 512x512px (untuk quality terbaik)
- **Aspect Ratio**: Square (1:1)

### Setelah Upload:
1. Pastikan file bernama **archa-hitam.png** (huruf kecil semua)
2. Letakkan di folder `public/`
3. Restart development server: `npm run dev`
4. Refresh browser
5. Logo akan muncul di tab browser dan bookmark

### Catatan:
- Logo akan otomatis muncul di:
  - Browser tab (favicon)
  - Bookmark
  - Mobile home screen (jika web di-save)
  - Share preview
