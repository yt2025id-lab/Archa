# Recording Checklist

## Screens to Record:
1. [ ] Landing page hero section (scroll slowly)
2. [ ] About section with problem/solution
3. [ ] How It Works section
4. [ ] Pools page - show all pools
5. [ ] Pool detail page - show join flow
6. [ ] Connect wallet modal
7. [ ] Approve USDC transaction
8. [ ] Join pool success
9. [ ] AI Optimizer page - full dashboard
10. [ ] Risk tolerance switching
11. [ ] Protocol list with APY
12. [ ] AI Recommendation card
13. [ ] GitBook documentation (quick)
14. [ ] GitHub README (quick)

## Tips:
- Use a clean browser (no bookmarks bar)
- Set browser zoom to 90% for more content
- Use smooth scrolling
- Pause 2-3 seconds on important elements
- Record at 1080p or higher
- Add subtle background music (lo-fi or corporate)

## Voice Over Tips:
- Speak clearly and with enthusiasm
- Pace: ~150 words per minute
- Add pauses for emphasis
- Record in a quiet room
- Use a good microphone
